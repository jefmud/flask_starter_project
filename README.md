# flask_starter_project
A flask starter project with user login and registration (compatible with Python 2.7.9 and higher)

Requirements (and dependencies)

The major libraries are:
1. Flask
2. Flask-admin
3. Flask-login
4. Flask-bcrypt
5. Flask-wtf
6. Peewee
7. wtf-peewee
